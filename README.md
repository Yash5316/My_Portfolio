## Installation
To set up the project locally, follow these steps:

1. Clone the repository:
    ```sh
    git clone https://github.com/Yash5316/My_Portfolio.git
    ```
2. Navigate to the project directory:
    ```sh
    cd My_Portfolio
    ```
3. Install the dependencies:
    ```sh
    npm install
    ```
